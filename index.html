3 Evaluación de Riesgos

Riesgos identificados con su probabilidad, impacto y estrategia de mitigación.

Error en credenciales JSON
Prob: Media Impacto: Alto
Verificar que serviceAccountKey.json está en cada carpeta backend/
PC sin Node.js instalado
Prob: Media Impacto: Alto
Instalar Node.js LTS desde nodejs.org antes de ejecutar
Conflicto de puertos locales
Prob: Media Impacto: Medio
Cambiar el PORT con variable de entorno al correr cada servidor
Cuota Firestore agotada
Prob: Baja Impacto: Alto
Monitorear en Firebase Console → Uso y facturación
Cambio de API de Firestore
Prob: Muy Baja Impacto: Medio
Fijar versión de firebase-admin en package.json
Sin conexión a internet
Prob: Baja Impacto: Medio
Usar Firebase Emulator Suite para desarrollo offline
